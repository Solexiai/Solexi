Overview

Solexi.ai is designed to protect sensitive legacy materials through layered controls: encryption, strong authentication, least-privilege access, and auditable events.

Threat Model (Plain English)

  • Account takeover: mitigated with MFA + login alerts + rate limiting.
  • Unauthorized heir access: mitigated with explicit permissions + verification steps.
  • Data loss: mitigated with backups + redundancy + recovery testing.
  • Insider risk: mitigated with least-privilege access + audit logs.
